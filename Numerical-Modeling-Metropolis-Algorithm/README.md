# Numerical-Modeling-Metropolis-Algorithm

In this specific project I have aimed to implement a numerical simulation using the Metropolis-Hastings Monte Carlo algorithm to calculate the parameters of a classical limit system. The code systematically samples the configuration space, computes the average potential energy, average square energy and heat capacity of the system, verifying the results against theoretical frameworks like the Equipartition Theorem.For simple systems, we can do this on paper. But for complex gases or molecular networks, the math becomes impossible.


The Problem given: We have a tiny particle attached to a microscopic spring (a 1D harmonic oscillator) sitting in a warm room at 300 K. We want to know two things about it: its Average Energy and its Heat Capacity (how much energy it needs to change temperature).

Project successfully demonstrates how Python and numerical modeling (Monte Carlo methods) can be used to bypass impossible calculus and accurately predict the physical and thermal behavior of complex systems.
